General Description
  Library for choosing the right film for the right day
  This is a pet project for developing my django ver 4.1 and some other skills
  It has to be a site for advanced films filtering



System requirements:
  python ( some of latest versions usually are better)
  in cmd( windows):
    pip install django
    pip install pillow
    pip install django-taggit
    pip install django-crispy-forms
    
To run this site on your computer:
  Navigate in cmd to project folder, where file manage.py is located. 
  Type'  python manage.py runserver' end press enter
  Open http://127.0.0.1:8000/ in your browser
